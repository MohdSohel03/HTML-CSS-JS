# 🤖 Tic-Tac-Toe AI Game

A modern, animated **Tic-Tac-Toe** game built using **HTML, CSS, and JavaScript** with two exciting game modes:

- 🎮 Player vs Player
- 🧠 Player vs Computer (AI using Minimax algorithm)

---

## 🕹️ Features

- 🎯 Two Game Modes: PvP & PvC
- 🧠 Smart AI using the Minimax algorithm
- 🏆 Score tracking for both players (X and O)
- ✨ Winning animations and hover effects
- 🔁 Restart button for a new round
- ♻️ Reset Scores button
- 🌙 Beautiful dark-themed responsive UI

## 🎮 How to Play
1. Player vs Player Mode:
    Click "Player vs Player"
    Player X and O take turns
    First to win gets a point!

2. Player vs Computer Mode:
    Click "Player vs Computer"
    You play as X, AI plays as O
    AI uses Minimax for unbeatable strategy

🟩 Winner tiles will blink on win!
🔁 Use Restart to start a new game
♻️ Use Reset Scores to clear the scoreboard

## 🛠 Technologies Used
HTML5 – Structure
CSS3 – Styling, grid layout, animations
Vanilla JavaScript – Game mechanics, DOM interaction, AI logic

🧠 Minimax Algorithm
The AI uses the classic Minimax algorithm to calculate the best move every turn, ensuring that:

It never loses (perfect play)
It will either win or draw
